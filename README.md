# al3aql-almodabber

Telegram bot that scans:
- 🏦 US stocks under $7 using Finnhub API
- 💰 Top 10 cryptocurrencies using CoinGecko

### Commands:
- `/scan_stocks` – Lists US stocks under $7
- `/scan_crypto` – Lists top 10 crypto coins

### Powered by:
- Finnhub (stocks)
- CoinGecko (crypto)
- Python + Flask + Telegram Bot
